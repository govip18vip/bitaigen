## P2P交易是什麼？

P2P是「Peer to Peer」（點對點）。不通過交易所，而是個人和個人直接交易。

**傳統交易所：**
你 → 交易所 → 別人

**P2P交易：**
你 ↔ 別人（交易所作為擔保方）

簡單說，就像你在蝦皮賣東西一樣。買家和賣家直接談定價格和交易方式，平台負責確保雙方誠實。

## P2P vs 交易所現貨：誰更便宜？

| 項目 | 交易所現貨 | P2P交易 |
|------|-----------|--------|
| **手續費** | 0.1-0.2% | 0-0.5%（可議） |
| **價格** | 市場價 | 可議價 |
| **速度** | 秒到 | 1-3小時 |
| **流動性** | 超高 | 有限 |
| **詐騙風險** | 幾乎無 | 中等 |

P2P的優點是**價格可以便宜**。比如現在市場ETH 50000台幣，你用P2P可能買到49500。看起來省500元，但考慮時間成本可能不值。

## 台灣常見的P2P平台

**1. 幣安P2P**
- 最大、最規範
- 用戶最多
- 審核嚴格

**2. OKX P2P**
- 功能完整
- 保護機制好
- 台灣使用者增加中

**3. 其他小平台**
- 某些本地P2P
- 風險更高
- 不建議新手用

我推薦幣安P2P，因為最安全。

## 幣安P2P操作步驟

### 第一步：進入P2P頁面

1. 進入幣安官網
2. 點「買幣」→「P2P交易」
3. 選「買」（購買加密貨幣）或「賣」

### 第二步：選擇交易者

假設你要買1萬元的USDT：

1. 輸入金額「10000」（台幣）
2. 系統會列出所有願意賣USDT的商家
3. 看價格、評分、回應時間

**怎麼選？**
- **價格**：越便宜越好
- **評分**：4.8以上最好
- **回應時間**：1分鐘內最佳
- **完成交易數**：超過1000筆比較放心
- **支付方式**：銀行轉帳最安全

### 第三步：發起交易

1. 點選一個賣家（比如「Alice」，評分4.95，銀行轉帳）
2. 輸入金額「10000」
3. 點「購買」
4. 確認交易單號

### 第四步：等待賣家確認

賣家會看到你的購買訂單。他會點「確認已收到付款」或「已發送資金」。

如果他選「已發送資金」，系統會立刻凍結他的USDT。現在，你就有15分鐘時間轉帳給他。

### 第五步：轉帳給賣家

賣家會給你銀行帳戶資訊。用你的銀行App轉帳：

1. 打開銀行App
2. 轉帳到賣家提供的帳戶
3. 金額要對（10000元）
4. **轉帳備註不要寫「加密貨幣」或「USDT」**（銀行會盯）

通常寫個模糊的備註，比如「朋友轉帳」或「購物」。

### 第六步：確認轉帳

1. 轉帳完成後，在P2P訂單頁面點「已轉帳」
2. 稍等一會兒，賣家會確認收到
3. 一旦賣家點「確認已收到」，USDT會直接進你的P2P帳戶
4. 你可以把USDT提現到自己的現貨帳戶

### 第七步：完成交易

現在USDT在你的現貨帳戶裡了。交易完成。整個過程通常1-2小時。

## P2P交易的安全提示

### 必做項：身份驗證

- 確認對方是驗證商家（有藍色勾勾）
- 看他的交易歷史和評分
- 看完成交易數（建議1000+）

### 轉帳時要小心

1. **確認金額**：再檢查一次
2. **確認帳戶**：最好讓對方讀出帳號最後4碼確認
3. **備註要隱晦**：不要寫「USDT」、「加密貨幣」
4. **不要操之過急**：轉帳後不用立刻點「已轉帳」，先確認錢確實出去了

### 防詐騙紅旗

- 對方要求你先轉帳再進訂單（假的）
- 要求用第三方平台轉帳（WeChat Pay等）
- 價格異常便宜（便宜50%以上）
- 評分超低但交易數超多（可能是刷的）
- 一直問你私人資訊

## P2P買幣被騙了怎麼辦？

假設對方說已轉帳，但你的銀行帳戶沒收到。怎麼辦？

**第一步：馬上申請仲裁**
在P2P訂單頁面點「申請仲裁」。不要點「確認已收到」。

**第二步：提供證據**
- 你銀行帳戶的轉帳明細
- 截圖對方說「已轉帳」的訊息
- 時間線

**第三步：等待幣安判決**
通常1-3天。如果證據足夠，幣安會把USDT原路退回給賣家。你拿回交易金額（不是USDT，是台幣）。

**成功率多少？**
大概80-90%。因為幣安有完整的交易紀錄，很容易判斷誰在說謊。

## 安全檢查清單

每次P2P交易前：

- [ ] 對方是驗證商家
- [ ] 評分4.8以上
- [ ] 完成交易數1000+
- [ ] 支付方式是銀行轉帳
- [ ] 價格合理（不會便宜超過2%）
- [ ] 金額確認無誤
- [ ] 轉帳時不涉及私人資訊

## 常見問題

**「P2P買的幣能直接轉出去嗎？」**
不能。P2P買的幣在P2P帳戶裡。要先轉到現貨帳戶，才能進一步操作。

**「P2P有手續費嗎？」**
幣安P2P本身不收手續費（賣家有時會內含手續費在價格裡）。但銀行轉帳要收轉帳手續費（5-10元）。

**「能用信用卡P2P買幣嗎？」**
某些P2P商家接受信用卡，但風險更高。建議用銀行轉帳。

**「用多個帳號P2P會被凍結嗎？」**
不會。但如果有異常交易模式（比如每天大額進出），可能被盯。正常使用沒問題。

## 重點提醒

> P2P交易雖然便宜，但風險更高。詐騙、銀行凍結等都有可能。新手建議先用中心化交易所熟悉，再進階P2P。P2P只適合有經驗的使用者。無論如何，安全永遠是第一位。貪小便宜反而容易被騙。
