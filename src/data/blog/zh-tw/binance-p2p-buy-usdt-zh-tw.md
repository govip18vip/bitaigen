## 為什麼要用P2P買USDT？

1. **便宜**：省2-5%的手續費
2. **靈活**：可議價，找到合適商家
3. **隱私**：不需要上傳身份證（某些商家）
4. **快速**：1-2小時完成

但也有風險：如果對方騙你說已轉帳其實沒轉，你要申請仲裁。所以選商家很重要。

## 台灣P2P買USDT完整流程

### 準備工作

1. 已完成幣安帳號註冊和KYC
2. 銀行帳號準備好
3. 決定買多少（建議第一次1000-5000元試試）

### 第一步：進入幣安P2P頁面

[進入幣安官網](/go/binance)，登入後：

1. 點頂部「買幣」
2. 選「P2P交易」
3. 確保在「買」頁面（不是「賣」）

### 第二步：設定購買金額

1. 左上方選「台幣 TWD」（貨幣）
2. 輸入金額（比如「5000」）
3. 選擇USDT（幣種）
4. 點「搜尋」

系統會列出所有願意用台幣賣USDT給你的商家。

### 第三步：選擇商家（最關鍵）

看到商家列表了，現在要選一個可靠的。看這幾個指標：

**1. 商家等級（必須藍勾）**
- 有藍色勾勾 = 驗證商家，相對安全
- 沒勾 = 普通用戶，風險更高
- 新手只選有勾的

**2. 評分（最少4.8以上）**
- 4.95-5.0 = 優秀
- 4.8-4.95 = 良好
- 低於4.8 = 不建議

**3. 完成交易數（至少500+）**
- 1000+ = 信得過
- 500-1000 = 可接受
- 100以下 = 新手賣家，風險較高

**4. 支付方式**
- **銀行轉帳**（最安全，推薦）
- WeChat Pay（高風險，對方可能是詐騙集團）
- 其他電子支付（風險中等）

**5. 自動釋放（可選項）**
- 有「自動釋放」勾勾的商家更安全
- 代表到期時間內自動確認，不會卡住

**5. 價格**
- 比市場便宜2-5%是正常
- 便宜超過5%以上要小心（可能是詐騙）

**綜合建議：**
選一個：藍勾 + 4.95分 + 1000+交易 + 銀行轉帳 + 價格合理 的商家。

### 第四步：點進商家詳情

點選一個符合條件的商家。會看到：
- 他的交易紀錄
- 用戶評價
- 響應時間
- 支付方式細節

簡單看一下評價，沒有紅旗就可以進行。

### 第五步：輸入購買金額並發起交易

1. 輸入你要買的金額（比如5000元）
2. 系統會顯示你能買到的USDT數量
3. 點「購買USDT」

會跳出確認畫面。再確認一次金額和USDT數量，沒問題就「確認」。

現在，訂單成立了。系統會給你一個訂單號。

### 第六步：等待賣家確認

賣家會在1-5分鐘內看到你的訂單。他會點「已發送資金」或確認。

一旦他確認，他的USDT會被系統凍結。現在，你有**15分鐘時間轉帳給他**。

### 第七步：聯絡賣家並轉帳

賣家會給你他的銀行帳戶資訊。通常在P2P介面的「聊天」區域。

**轉帳資訊會包括：**
- 銀行名稱
- 帳戶姓名
- 帳戶號碼
- 金額

**現在打開你的銀行App：**

1. 點「轉帳」
2. 選「轉帳到其他銀行」（如果是同行選「行內轉帳」）
3. 輸入帳戶號碼
4. 輸入金額（5000元）
5. **備註寫什麼？** 寫個模糊的備註，比如「朋友轉帳」或什麼都不寫
6. 點「確認」
7. 完成轉帳

**注意事項：**
- 不要寫「USDT」、「加密」、「虛擬貨幣」等關鍵字
- 銀行AI會盯，寫出來可能被凍帳
- 「朋友轉帳」、「購物」、「代購」都可以

### 第八步：確認轉帳並等待賣家確認

1. 回到P2P訂單頁面
2. 點「已轉帳」（代表你已經轉帳了）
3. 等賣家確認收到

這時候，要看賣家多久會確認。通常1-5分鐘。

如果賣家點「確認已收到」，系統就會把USDT釋放給你。你的P2P帳戶裡就有USDT了。

### 第九步：提現到現貨帳戶（可選）

USDT現在在「P2P帳戶」裡。如果你要交易或轉出，需要轉到「現貨帳戶」：

1. 點「資產」→「P2P」
2. 看USDT餘額
3. 點「轉帳」
4. 轉到「現貨帳戶」
5. 確認

秒到。現在你就有USDT在現貨帳戶裡了。

## 如果交易出問題怎麼辦？

### 情況1：賣家說收不到轉帳

**可能原因：**
- 你轉帳確實失敗了（銀行拒絕）
- 轉帳到錯誤帳戶
- 金額不對

**怎麼做：**
1. 查你的銀行交易紀錄
2. 確認轉帳是否成功
3. 如果成功，截圖轉帳紀錄給賣家看
4. 如果失敗，再轉一次

### 情況2：賣家一直不確認收到

**可能原因：**
- 賣家故意拖延（想詐騙你）
- 賣家忘記確認

**怎麼做：**
1. 等待10分鐘
2. 在聊天區催促他
3. 如果超過15分鐘還沒確認，立刻申請仲裁

### 情況3：申請仲裁

在P2P訂單頁面，點「申請仲裁」。提供：

1. 你銀行轉帳的截圖
2. 轉帳時間和金額
3. 對方帳戶資訊
4. 所有聊天紀錄

幣安會在1-3天內判決。大多數情況會判你贏，因為有銀行轉帳紀錄作證。

## 台灣用戶常見問題

**「銀行會凍帳嗎？」**
不會，只要：
1. 不寫「加密」關鍵字
2. 金額不超大（建議每筆5000-10000元）
3. 不頻繁轉帳（正常用戶每月1-2次）

正常使用不會被凍。

**「可以用別人帳戶轉帳嗎？」**
可以，但要確認帳戶姓名和P2P註冊名字匹配。如果不匹配，賣家可能拒絕或申請仲裁。

**「第一次買多少最安全？」**
建議1000-5000元。這樣即使被騙也不會太虧。確認流程沒問題後，再加大金額。

## 重點提醒

> P2P是便宜買幣的方式，但風險更高。選擇信得過的商家、正確的轉帳方式、合理的價格，就能大幅降低風險。如果有任何懷疑，就直接用現貨購買，多花2-3%手續費換得安心。本文僅提供教育用途，購買加密貨幣前請確保了解風險。
