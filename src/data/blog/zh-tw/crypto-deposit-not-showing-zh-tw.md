## 排查清單

入金沒到帳，按順序檢查：

### 1. 確認是否已發送

檢查你的銀行帳戶：
- 交易是否已扣款
- 轉帳記錄中是否有該筆交易
- 帳戶備註是否填對了（UID）

### 2. 檢查交易狀態

在銀行或支付平台查看：
- 交易是否「已完成」
- 轉帳金額是否正確
- 目標帳戶是否正確

### 3. 等待足夠時間

- 銀行轉帳：1-2工作天
- 週末/假日：可能延遲到下週
- 大額轉帳：銀行可能審查，需要更長時間

### 4. 檢查區塊鏈

如果是加密貨幣轉帳（不是銀行轉帳）：

1. 在幣安複製你的充值地址
2. 訪問區塊瀏覽器（etherscan.io等）
3. 搜索該地址
4. 查看是否有該筆交易

**確認項目**：
- [ ] 交易狀態：「成功」或「確認中」
- [ ] 確認數：至少6個確認（通常需要）
- [ ] 金額：是否匹配

### 5. 聯絡交易所客服

如果以上都沒問題，聯絡幣安客服：

**提供信息**：
- 充值地址
- 轉帳金額
- 轉帳時間
- 交易Hash（如有）
- 交易所帳號

**等待時間**：通常24-48小時回覆

## 常見問題和解決

**問題1：銀行說已轉帳，但交易所查不到**

→ 確認目標帳戶是否正確。如果是小銀行，可能需要人工處理。

**問題2：轉帳時沒填UID**

→ 聯絡客服，提供轉帳記錄和金額。客服可以手動查找。

**問題3：誤轉到別人帳戶**

→ 立即聯絡銀行和交易所。如果是同一交易所，客服可能能恢復。

**問題4：區塊鏈上有交易，但交易所查不到**

→ 可能是轉帳地址錯誤（轉到了合約地址）。聯絡客服確認。

## 預防措施

1. **第一次轉小額**

   先轉1000元測試，確認無誤再轉大額。

2. **檢查三次**

   複製地址時檢查3遍，避免誤轉。

3. **保存記錄**

   截圖轉帳記錄和區塊確認，以備客服查詢。

4. **及時反映**

   如果發現問題，立即聯絡，不要等太久。

## 聯絡客服的技巧

1. **清楚描述問題**

   「我在2026-04-11 10:00轉帳10,000 USDT到地址0x...，到現在還沒到帳」

2. **提供完整信息**

   交易Hash、時間、金額、轉帳地址，一次給全。

3. **有耐心**

   客服回應可能需要24-48小時，不要急。

4. **保持禮貌**

   禮貌的客戶優先級更高。

## 總結

99%的入金遲到問題可以通過等待、檢查區塊和聯絡客服解決。放心，你的錢不會丟。
