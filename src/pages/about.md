## 关于 Bitaigen

**Bitaigen** 是一个专业的多语言区块链资讯平台，致力于为全球加密货币投资者提供高质量的行情分析、交易所教程、钱包安全指南和 DeFi 深度解析。

我们的内容覆盖 **简体中文、繁體中文、English、Español、Português** 五种语言，服务全球华语、英语、西班牙语和葡萄牙语用户。

## 技术栈

Bitaigen 基于 [Astro](https://astro.build/) 构建，采用静态站点生成(SSG)技术，确保极致的加载速度和 SEO 表现。

- **框架**: Astro + TypeScript
- **样式**: Tailwind CSS
- **搜索**: PageFind (静态全文搜索)
- **行情数据**: CoinGecko API
- **部署**: Cloudflare Pages
